# Evaluation rubric — Every Tongue study-material drafts

Each generated output is scored on five criteria (1 = poor, 5 = excellent).
The rubric is deliberately weighted toward the failure modes that matter most
for ministry use (Scripture fidelity and the review-flagging that catches
low-resource weakness).

| # | Criterion | What a 5 looks like |
| --- | --- | --- |
| 1 | **Scripture fidelity** | Quotes the target-language translation **exactly**; no fabricated, drifted, or invented Scripture. (Weighted highest — this is the doctrinal failure mode.) |
| 2 | **Target-language fluency** | Grammatical, idiomatic, correct register for the audience; not repetitive or garbled. |
| 3 | **Structure adherence** | All required sections for the material type are present and in order. |
| 4 | **Review-flag quality** | The "needs native-speaker review" flags surface the genuinely uncertain phrases (especially in low-resource languages). |
| 5 | **Back-translation faithfulness** | The English back-translation honestly reflects the target-language draft so a non-speaker can verify meaning. |

## Scoring protocol

- Scores require a **native/fluent speaker** of the target language for criteria
  2 and 4. Where no qualified reviewer has scored a case, it stays **pending**.
- Scripture fidelity (1) can be checked mechanically against the ScriptureFlow
  source text.
- **No score is recorded unless it was actually assessed.** See `results.md`.
